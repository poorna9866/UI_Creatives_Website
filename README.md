## 1) Project Overview  
The **UI Creatives Website** is a creative, responsive 3-page website built using **Bootstrap 5** and **CSS3**.  
The goal was to design a clean, modern, and professional layout that adapts smoothly to all screen sizes without using JavaScript.  

**Pages Included:**  
- **Home Page:** Navbar, hero banner, features, carousel, footer  
- **About Page:** Mission, Vision, Team sections  
- **Contact Page:** Form and social links
